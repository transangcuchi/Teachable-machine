Thư viện cần cài: 
keras.models 
    pip install keras-models
opencv
    pip install opencv-python
numpy
    pip install numpy
TensorFlow
    pip install TensorFlow
tkinter
    pip install tkinter
Pillow
    pip install Pillow

Hướng dẫn sử dụng:
    Cài các thư viện
    Chạy chương trình
    Ấn vào nút "Nhận Diện" để sử dụng máy ảnh máy để quét
        Ấn ESC ở bàn phím trong khi đưa vật vào máy ảnh
        Xuất kết quả ở giao diện
    Ấn vào nút "Tải ảnh lên" để dùng ảnh, Ảnh sử dụng nên là .jfif hoặc là .jpg.
        Xuất kết quả ở giao diện
